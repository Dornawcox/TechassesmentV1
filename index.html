What is Ethical Technology Assessment?

A framework for evaluating whether agricultural digital tools align with the values and needs of farming communities.

Entry Gates

Pass/fail criteria every tool must meet:

  • Documentation publicly accessible
  • Users retain data rights
  • No surveillance or abuse potential
  • Responsible stewardship

Scored Dimensions

Graduated assessment (0-25 points each):

  • Data Ethics — consent, privacy, minimization
  • Interoperability — APIs, standards, portability
  • Commons Fit — license, governance, contribution
  • Accessibility — languages, bandwidth, offline

Confidence Tiers

How thoroughly has it been reviewed?

  • ★☆☆ Self-reported / marketing claims
  • ★★☆ Peer-reviewed / community tested
  • ★★★ Verified by CGO steward
